Prepaid Energy Meter Using Arduino and GSM Module:
Aug 2019 - Dec 2019

In this project we built an automated system by using Arduino and GSM module. We can recharge the electricity balance through this system, just by sending a SMS. It can also disconnect the home power supply connection, if there is low or zero balance in the system and this system will also send some updates to user’s mobile phone like low balance alert, cut off alert and recharge alert.
